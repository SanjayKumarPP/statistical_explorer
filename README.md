# 📊 Statistical Explorer

An interactive Streamlit-based data science application for comprehensive statistical analysis, visualization, and hypothesis testing. This tool allows users to upload CSV or Excel datasets and instantly explore data through descriptive statistics, graphical insights, and statistical inference techniques.

---

## 🚀 Live Demo
👉 https://your-streamlit-app-link.streamlit.app

---

## 📌 Features

### 📂 Data Upload
- Upload CSV, XLSX, XLS datasets
- Automatic detection of numeric & categorical columns
- Instant dataset preview

### 📐 Descriptive Statistics
- Mean, Median, Mode
- Standard Deviation & Variance
- Range, IQR, CV
- Skewness & Kurtosis
- Confidence Intervals
- Percentiles (P1, P25, P50, P75, P99)

### 📈 Visualizations
- Histogram + KDE
- Box plot
- Q-Q plot
- Violin plot
- Empirical CDF
- Scatter plots
- Run charts

### 🔬 Hypothesis Testing
- Shapiro-Wilk, Anderson-Darling, D’Agostino-Pearson
- One-sample & two-sample t-tests
- ANOVA & Kruskal-Wallis
- Chi-square tests
- Correlation analysis (Pearson, Spearman, Kendall)

---

## 🧠 Objective
To simplify statistical analysis by providing an intuitive, no-code interface for data exploration, visualization, and hypothesis testing.

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/statistical-explorer.git
cd statistical-explorer
pip install -r requirements.txt
streamlit run statistical_explorer.py
```

---

## ☁️ Deployment
1. Push to GitHub
2. Go to https://share.streamlit.io/
3. Select repo and file `statistical_explorer.py`
4. Click Deploy
